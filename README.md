#  Synergy – Smart Health & Fitness Management System

## Overview

**Synergy** is a smart health and fitness management system designed to connect users with experts for personalized diet plans, workout routines, and real-time guidance. The platform integrates web and mobile technologies to deliver a seamless and interactive fitness experience.

---

##  Objectives

* Provide personalized diet and workout plans
* Enable real-time communication between users and experts
* Track user progress and performance
* Simplify health management through a centralized platform

---

##  Technologies Used

* **Frontend (Web):** HTML, CSS, JavaScript
* **Frontend (Mobile):** Flutter
* **Backend:** Django (Python)
* **Database:** MySQL
* **Tools:** GitHub, VS Code

---

##  Features

*  User Registration & Login
*  Expert Panel for managing users
*  Chat system between user and expert
*  Diet plan management
*  Workout tracking system
*  Progress monitoring
*  Secure authentication

---

##  System Architecture

The system follows a **client-server architecture**:

* Frontend interacts with backend APIs
* Django handles business logic
* MySQL manages data storage

##  Installation & Setup

### 1️ Clone the repository

```
git clone https://github.com/christoantu/Synergy.git
cd Synergy
```

### 2️ Install dependencies

```
pip install -r requirements.txt
```

### 3️ Run the server

```
python manage.py runserver
```

### 4️ Open in browser

```
http://127.0.0.1:8000/
```
